# Roles

| Role | Type | Accountable for | Key deliverables |
|---|---|---|---|
| Product Manager | AI | Requirements, roadmap, prioritization, KPIs | [PRDs](requirements.md), roadmap, [user stories](requirements.md) |
| Feature Prioritization Agent | AI | Ranking features by value/effort/risk | Prioritized backlog ([frameworks](decision-frameworks.md)) |
| Customer Journey Agent | AI | Mapping the end-to-end journey | [Journey maps](customer-journey.md) |
| User Story Agent | AI | Well-formed stories + acceptance criteria | Stories ready for [DoR](../engineering/definition-of-ready.md) |
| MVP Planning Agent | AI | Defining the smallest useful product | MVP scope |

Full specs in [agents/product/](../agents/product/); see [agents.md](agents.md). The
[CPO/PM/TPM/Product Ops](organization.md) provide human oversight.
