# User Personas

Each persona is grounded in [discovery](discovery.md) evidence, not invented.

## Persona template

- **Name / archetype**
- **Role**
- **Goals**
- **Responsibilities**
- **Pain points**
- **Current tools / workflow**
- **Technical expertise**
- **Buying influence**
- **Success metrics** (how they judge success)

## Usage

Personas anchor the [PRD](requirements.md), the [customer journey](customer-journey.md), and
[prioritization](decision-frameworks.md) — a feature that serves no persona's real job is a candidate
to cut. Personas are stored in [customer memory](../memory/customer-memory.md) where tied to real accounts.
