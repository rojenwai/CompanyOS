# Product Dashboards

## North-star dashboard
**Audience:** product + CPO · **Cadence:** daily
Panels: north-star metric · activation · retention · churn.

## Adoption dashboard
**Audience:** product · **Cadence:** per release + weekly
Panels: feature adoption · conversion funnel · [journey](customer-journey.md) drop-off.

## Experiment dashboard
**Audience:** product + data · **Cadence:** per experiment
Panels: variant performance · significance · decision status ([analytics](analytics.md)).

Metrics defined in [metrics.md](metrics.md); alerts route via [automation](automation.md).
