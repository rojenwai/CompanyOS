# Product Launch Checklist

The product view of launch readiness. Pairs with the engineering
[release checklist](../../templates/documents/release-checklist.md).

- [ ] Problem and target [persona](../user-personas.md) validated
- [ ] Success metric defined and **instrumented** ([analytics](../analytics.md))
- [ ] [PRD](../requirements.md) approved by product, engineering, design, security
- [ ] Engineering [Definition of Done](../../engineering/quality-gates.md) met
- [ ] QA passed; security approved; performance validated
- [ ] Documentation complete ([user + API](../../engineering/documentation-requirements.md))
- [ ] Support and on-call briefed ([post-launch](../../post-launch/support.md))
- [ ] Marketing / GTM prepared
- [ ] Rollback plan available ([rollback](../../post-launch/rollback.md))
- [ ] Post-launch review scheduled
