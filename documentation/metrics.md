# Documentation - Metrics

**North-star:** Readers find accurate answers fast - high doc coverage, freshness, and search success.

**Coverage** - feature doc coverage, API endpoint coverage

**Currency** - doc freshness, stale-doc count, time-to-update after change

**Usefulness** - search success rate, reader task success, doc satisfaction

Metrics are surfaced on the [dashboards](dashboards.md) and feed the
[Continuous Improvement Engine](../orchestration/continuous-improvement-engine.md).
