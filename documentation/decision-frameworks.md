# Documentation - Decision Frameworks

How decisions are made here. All decisions follow the company
[decision framework](../governance/decision-framework.md); these are the department-specific ones.

- **Fix vs. retire a stale doc** - Fix if still needed; retire if obsolete - never leave it misleading.
- **Generate vs. hand-write** - Generate reference docs from the source of truth; hand-write concepts and guides.
- **One doc vs. split** - One home per topic; split only when audiences or tasks genuinely differ.

Reversible decisions are made fast and revisited; irreversible ones require review and are recorded in
[decision memory](../memory/decision-memory.md).
