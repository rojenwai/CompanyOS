# Documentation - Agents

The Documentation agents, each written from the [agent template](../agents/agent-template.md) and living
in [agents/documentation/](../agents/documentation/).

| Agent | Mission (one line) |
|---|---|
| [Technical Writer Agent](../agents/documentation/technical-writer-agent.md) | Produce clear, accurate, task-oriented documentation for every product capability. |
| [API Documentation Agent](../agents/documentation/api-documentation-agent.md) | Keep API reference docs complete, accurate, and example-rich - generated from the source of truth. |
| [Knowledge Base Agent](../agents/documentation/knowledge-base-agent.md) | Keep internal knowledge - runbooks, processes, onboarding - findable, owned, and current. |

All agent output is reviewed per [review-process.md](review-process.md); major decisions escalate to the
[CEO Agent](../orchestration/ceo-agent.md).
