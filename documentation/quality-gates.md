# Documentation - Quality Gates

Work does not progress until the relevant gate passes. These implement the company
[quality gates](../governance/quality-gates.md).

| Gate | Must pass |
|---|---|
| Coverage gate | Every shipped capability is documented |
| Accuracy gate | Docs match current product behavior |
| Ownership gate | Every doc has an owner and freshness date |
| Findability gate | Content is structured and searchable |
