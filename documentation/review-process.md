# Documentation - Review Process

Documentation is reviewed by the [Documentation Reviewer](../orchestration/documentation-reviewer.md) for accuracy (does it match the product?), completeness (is anything missing?), and clarity (is it written for the reader?). Docs are a release gate.

Reviews are independent - the [Reviewer](../orchestration/reviewer.md) never assumes the author is
correct. Security-sensitive work also requires the
[Security Reviewer](../orchestration/security-reviewer.md).
