# Research - Playbooks

Situational guides. Each names the trigger and the steps.

### A promising problem has thin evidence

1. State what is known vs. assumed
2. Design the cheapest validating test
3. Run it
4. Re-score with the new evidence
5. Pursue, defer, or decline transparently

### A competitor ships something similar

1. Assess overlap and differentiation
2. Re-check defensibility
3. Update positioning with [strategy](../strategy/go-to-market.md)
4. Decide accelerate, differentiate, or exit
