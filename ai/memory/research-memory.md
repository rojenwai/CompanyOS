# Research Memory

**Scope:** the company's accumulated research, long-lived.

## What's stored
Research findings · benchmarks · sources · competitor profiles · experiment results — backed by [research/](../../handbook/departments/research/).

## Lifetime
Permanent; sources carry freshness metadata so stale findings are flagged.

## Writers / readers
Written by research agents as [research reports](../../handbook/templates/documents/research-report.md); read whenever a decision needs evidence.

## Rule
Every stored finding cites its source and distinguishes fact from assumption ([Evidence Over Assumptions](../../handbook/company/principles.md)).
