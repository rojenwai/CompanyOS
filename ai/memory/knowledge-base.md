# Knowledge Base

**Scope:** durable reference knowledge, long-lived.

## What's stored
Scientific concepts · algorithms · framework documentation · reference material — the factual substrate agents retrieve from before generating.

## Lifetime
Permanent; sources are refreshed and reliability-scored.

## Writers / readers
Curated by the [Knowledge Manager](../orchestration/knowledge-manager.md); read via [RAG](../../handbook/departments/ai-engineering/README.md) whenever factual accuracy matters.

## Retrieval contract
Every retrieval returns **source, confidence, timestamp, and relevance score** — see [retrieval.md](retrieval.md).
