# Handbook — the reading side

**Everything a human reads.** This is the company operating manual: identity, governance, the twenty
department manuals, shared workflows and standards, playbooks, templates, and guides.

The machine side lives in [ai/](../ai/README.md) — agent specs, orchestration, and memory. Keep the two
separate: **edit the handbook to describe how *your* company works; edit `ai/` to change how the agent
workforce behaves.** See [CUSTOMIZE.md](../CUSTOMIZE.md) to make this yours.
