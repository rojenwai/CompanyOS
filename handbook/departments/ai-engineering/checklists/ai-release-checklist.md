# AI Release Checklist

- [ ] Evaluation suite passes (accuracy, hallucination, safety, latency, cost)
- [ ] Compared against current production version
- [ ] Grounding verified (source, confidence, timestamp)
- [ ] Prompt/model versioned
- [ ] Model card written
- [ ] Adversarial/injection tests pass
- [ ] Cost per request within budget
- [ ] Rollback tested
