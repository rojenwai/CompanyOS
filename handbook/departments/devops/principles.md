# DevOps - Principles

Non-negotiable. Every piece of work is judged against these.

1. **Everything as code.** Infrastructure, pipelines, and config are versioned and reviewed.
2. **Automate the path to production.** Manual deploys are a bug.
3. **Every release is reversible.** No deploy without a tested rollback.
4. **Immutable and reproducible.** Rebuild any environment from source.
5. **Progressive delivery.** Canary and phased rollouts over big-bang.
6. **You build it, you run it - with a platform that helps.** DevOps paves the road; teams drive.
7. **Least privilege in infrastructure.** Secrets in a vault, access scoped and audited.
