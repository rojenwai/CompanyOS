# DevOps - Standard Operating Procedures

### SOP-01 - Cut a release

1. Confirm all gates pass
2. Tag the version (semver)
3. Deploy canary and verify
4. Progressive rollout
5. Publish release notes
6. Monitor post-deploy

### SOP-02 - Onboard a service to CI/CD

1. Add pipeline-as-code
2. Wire quality and security gates
3. Define environments
4. Add rollback
5. Add monitoring and alerts
6. Document the runbook
