# Investor Relations - Workflows

The core repeatable processes this department runs.

### Send the investor update

1. Pull reconciled metrics and milestones
2. Draft highlights, lowlights, and asks
3. Reconcile every number to [finance](../finance/README.md)
4. Get CEO/CFO approval
5. Distribute and update the data room

### Run a board meeting

1. Set the agenda and pre-read
2. Distribute materials early
3. Hold the meeting
4. Record minutes and resolutions
5. Track action items to closure
