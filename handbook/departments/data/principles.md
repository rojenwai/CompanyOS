# Data - Principles

Non-negotiable. Every piece of work is judged against these.

1. **One source of truth per metric.** Defined once, used everywhere.
2. **Data quality is tested,** like code - freshness, completeness, and validity checks.
3. **Privacy by design.** Collect the minimum; protect and govern the rest ([security](../security/README.md)).
4. **Reproducible pipelines.** As code, versioned and observable.
5. **Document the meaning.** Every metric has a definition and an owner.
6. **Access is least-privilege.** Especially for sensitive data.
