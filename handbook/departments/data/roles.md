# Data - Roles

| Role | Accountable for |
|---|---|
| Head of Data (CTO domain) | Data strategy, governance, metric definitions |
| Data Engineer | Pipelines, warehouse, data quality |
| Analytics Engineer | Data modeling, the metrics layer |
| Data Scientist / Analyst | Insight, experimentation, predictive models |

Human roles are amplified by the [department agents](agents.md). Every role follows the
[principles](principles.md) and the [review process](review-process.md).
