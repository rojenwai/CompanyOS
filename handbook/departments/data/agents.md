# Data - Agents

The Data agents, each written from the [agent template](../../../ai/agents/agent-template.md) and living
in [agents/data/](../../../ai/agents/data/).

| Agent | Mission (one line) |
|---|---|
| [Data Engineer Agent](../../../ai/agents/data/data-engineer-agent.md) | Build reliable, reproducible pipelines that land trustworthy data in the warehouse. |
| [Analytics Engineer Agent](../../../ai/agents/data/analytics-engineer-agent.md) | Model raw data into a governed metrics layer that everyone can trust and reuse. |
| [Data Scientist Agent](../../../ai/agents/data/data-scientist-agent.md) | Turn data into insight and predictive models that drive decisions. |
| [Data Governance Agent](../../../ai/agents/data/data-governance-agent.md) | Keep data private, secure, well-cataloged, and correctly retained. |

All agent output is reviewed per [review-process.md](review-process.md); major decisions escalate to the
[CEO Agent](../../../ai/orchestration/ceo-agent.md).
