# Data - Vision

- Horizon 1 - Reliable pipelines and a single, defined set of core metrics (no dueling dashboards).
- Horizon 2 - Self-serve analytics with tested data quality and a governed catalog.
- Horizon 3 - Real-time, privacy-preserving data powering product, AI, and the business by default.

The end state: everyone trusts the numbers, and no one argues about whose are right.
