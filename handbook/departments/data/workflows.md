# Data - Workflows

The core repeatable processes this department runs.

### Deliver a new metric/dataset

1. Capture the definition and owner
2. Build/extend the pipeline with tests
3. Model it in the metrics layer
4. Document in the catalog
5. Publish for self-serve
6. Monitor quality

### Run an experiment analysis

1. Confirm the hypothesis and metric
2. Verify data quality and power
3. Analyze with sound method
4. Report effect and uncertainty
5. Feed the decision
