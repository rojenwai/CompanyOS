# Data - Standard Operating Procedures

### SOP-01 - Add a data source

1. Assess quality and licensing/privacy
2. Build ingestion with tests
3. Classify sensitivity
4. Model and document
5. Publish and monitor

### SOP-02 - Define a metric

1. Agree the definition and owner
2. Model once in the semantic layer
3. Test it
4. Catalog the definition
5. Retire conflicting versions
