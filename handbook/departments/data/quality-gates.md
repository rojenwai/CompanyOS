# Data - Quality Gates

Work does not progress until the relevant gate passes. These implement the company
[quality gates](../../governance/quality-gates.md).

| Gate | Must pass |
|---|---|
| Quality gate | Freshness, completeness, and validity tests pass |
| Definition gate | Metric defined once, owned, and cataloged |
| Governance gate | Sensitive data classified and access-controlled |
| Method gate | Analyses use sound, reproducible method |
