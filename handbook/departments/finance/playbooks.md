# Finance - Playbooks

Situational guides. Each names the trigger and the steps.

### Runway drops below threshold

1. Recompute runway to the week
2. Model cut, raise, and revenue scenarios
3. Prioritize spend by value
4. Decide with the CEO/CFO
5. Communicate to the board via [investor-relations](../investor-relations/README.md)

### Revenue misses forecast

1. Diagnose the driver (acquisition, churn, expansion)
2. Update the forecast honestly
3. Model the runway impact
4. Align with [strategy](../strategy/README.md) on response
5. Adjust budget
