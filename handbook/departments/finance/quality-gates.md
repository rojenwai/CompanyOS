# Finance - Quality Gates

Work does not progress until the relevant gate passes. These implement the company
[quality gates](../../governance/quality-gates.md).

| Gate | Must pass |
|---|---|
| Accuracy gate | Statements reconcile; numbers trace to sources |
| Transparency gate | Forecast assumptions documented |
| Approval gate | Commitments have the required sign-off |
| Runway gate | Runway impact assessed before spend |
