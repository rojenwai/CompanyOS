# Post-Launch Tools

| Category | Purpose | Notes |
|---|---|---|
| Monitoring & alerting | Detect issues fast | Severity-based routing ([monitoring](monitoring.md)) |
| Observability | Metrics, logs, traces | No secrets in logs ([observability](observability.md)) |
| Incident management | Coordinate response | [Incident report](../../templates/documents/incident-report.md) template |
| Error/crash reporting | Surface production errors | Feeds [bug-management](bug-management.md) |
| Dependency & security scanners | Find vulnerable/outdated deps | Feeds patch agents |
| Backup & restore | Durable data | Restores tested, not assumed |
| Cost management | Track/limit spend | Cost [dashboards](dashboards.md) |
| Status page | Communicate to customers | Used during [incidents](incident-response.md) |

Tool choices follow the company [decision framework](../../governance/decision-framework.md).
