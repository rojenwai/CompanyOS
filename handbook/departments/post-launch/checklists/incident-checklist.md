# Incident Checklist

During a live [incident](../incident-response.md):

- [ ] Alert acknowledged within the [severity](../monitoring.md) target
- [ ] Incident declared; Incident Commander assigned
- [ ] Impact assessed (customers, services, business)
- [ ] Stakeholders/customers notified (status page)
- [ ] Mitigation applied ([rollback](../rollback.md) / [hotfix](../hotfix-policy.md))
- [ ] Service restoration verified in [observability](../observability.md)
- [ ] Incident closed; [incident report](../../../templates/documents/incident-report.md) started
- [ ] Postmortem scheduled; action items assigned
- [ ] Lessons filed to [lessons-learned](../../../../ai/memory/lessons-learned.md)
