# Research - Standard Operating Procedures

### SOP-01 - Run a research study

1. Define the question and success criteria
2. Choose the method
3. Gather sources and log each with date and confidence
4. Analyze and form a hypothesis
5. Write a [research report](../../templates/documents/research-report.md)
6. Persist to [research memory](../../../ai/memory/research-memory.md)

### SOP-02 - Score an opportunity

1. Assemble the evidence bundle
2. Score each criterion with rationale
3. Run a sensitivity check
4. Recommend pursue/defer/decline
5. Submit to the review board
