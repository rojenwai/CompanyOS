# Research - Quality Gates

Work does not progress until the relevant gate passes. These implement the company
[quality gates](../../governance/quality-gates.md).

| Gate | Must pass |
|---|---|
| Evidence gate | Every claim sourced, dated, and confidence-rated |
| Significance gate | The problem is shown to be frequent, severe, and urgent |
| Scoring gate | Opportunity scored on all criteria with written rationale |
| Board gate | Significant bets pass the Innovation Review Board |
