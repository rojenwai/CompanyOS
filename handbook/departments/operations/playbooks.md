# Operations - Playbooks

Situational guides. Each names the trigger and the steps.

### A project is slipping

1. Re-baseline scope, owners, and dates
2. Identify the true blocker
3. Cut scope or add help - decide explicitly
4. Communicate the new plan
5. Track to the new baseline

### Recurring process is causing pain

1. Map and measure it
2. Find the waste and handoffs
3. Redesign lighter and automate
4. Document and roll out
5. Measure the improvement
