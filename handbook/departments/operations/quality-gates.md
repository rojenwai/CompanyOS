# Operations - Quality Gates

Work does not progress until the relevant gate passes. These implement the company
[quality gates](../../governance/quality-gates.md).

| Gate | Must pass |
|---|---|
| Plan gate | Initiative has owner, plan, and success criteria |
| Status gate | Status is current in the single source of truth |
| Decision gate | Key decisions recorded |
| Closeout gate | Learnings captured; access/vendor items reconciled |
