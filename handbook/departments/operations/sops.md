# Operations - Standard Operating Procedures

### SOP-01 - Plan an initiative

1. Define goal and success criteria
2. Break into work with owners and dates
3. Map dependencies and risks
4. Set the status cadence
5. Record decisions

### SOP-02 - Onboard a vendor/tool

1. Define the need and criteria
2. Evaluate build vs. buy
3. Negotiate terms
4. Provision least-privilege access
5. Track renewal and performance
