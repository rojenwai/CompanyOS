# Design - Roles

| Role | Accountable for |
|---|---|
| Chief Designer | Design vision, the design system, accessibility bar, final design review |
| UX Researcher | Studies of behavior, needs, and mental models; usability testing |
| Product / UX Designer | User flows, information architecture, wireframes, interactive prototypes |
| UI / Visual Designer | Visual language, layout, typography, color, iconography, component design |
| Design Systems Owner | The [design system](design-system.md) - components, tokens, versioning, docs |
| Accessibility Specialist | WCAG conformance, assistive-tech testing, remediation |

Human roles are amplified by the [design agents](agents.md). Each role follows the
[design principles](principles.md) and the [review process](review-process.md).
