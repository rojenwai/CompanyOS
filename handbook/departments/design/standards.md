# Design - Standards

Testable standards, not aspirations. These extend the company-wide
[standards](../../standards/README.md).

1. Every surface meets [WCAG](accessibility.md) - accessibility is a release gate.
2. Every UI is built from the [design system](design-system.md); no undocumented one-offs.
3. Every interactive element has visible feedback and a focus state.
4. Every screen designs its loading, empty, and error states.
5. Significant design changes are validated with [users](usability-testing.md) before build.
