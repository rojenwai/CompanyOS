# Design - Standard Operating Procedures

### SOP-01 - Hand off a design to engineering

1. Finalize designs with all states
2. Attach specs, tokens, and component references
3. Confirm accessibility annotations
4. Walk through interactions and edge cases
5. Stay available during build for questions

### SOP-02 - Add a design-system component

1. Validate the need is recurring
2. Design it accessible and consistent
3. Version and document usage
4. Publish and announce
5. Track adoption
