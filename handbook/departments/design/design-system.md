# Design System

The single source of truth for the product's visual and interaction language, so every surface feels
like one product.

## Contains

Typography · color palette · spacing · icons · buttons · inputs · cards · tables · navigation ·
modals · charts · motion · dark mode.

## Rules

- Components are reusable and themeable; [frontend](../../../ai/agents/engineering/frontend-engineer.md) builds to them.
- Every component meets [accessibility](accessibility.md) requirements by default.
- Changes are versioned and reviewed; the system is documented in [documentation memory](../../../ai/memory/documentation-memory.md).

For charts and data visualization specifically, follow the dataviz guidance so visualizations read as
one system in both light and dark themes.
