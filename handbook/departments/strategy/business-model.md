# Business Model

## Possible models

SaaS · enterprise · marketplace · freemium · open core · licensing · hardware · hardware + SaaS ·
usage-based · subscription · advertising · developer platform · API · data-as-a-service · consulting ·
training · hybrid.

## Revenue analysis

Primary · secondary · future · recurring · transaction · platform · partnership · data revenue.

## Pricing strategy

**Pricing should reflect customer value, not development cost.** Evaluate: cost-plus · value-based ·
tiered · per user · per device · per API · per transaction · enterprise · custom.
