# Product AI Agents

Full specs in [agents/product/](../../../ai/agents/product/).

| Agent | Mission (one line) | Spec |
|---|---|---|
| Product Manager | Turn validated problems into prioritized, well-defined product work | [product-manager.md](../../../ai/agents/product/product-manager.md) |
| Feature Prioritization Agent | Rank features by evidence, not opinion | [feature-prioritization-agent.md](../../../ai/agents/product/feature-prioritization-agent.md) |
| Customer Journey Agent | Map the full customer journey to find friction | [customer-journey-agent.md](../../../ai/agents/product/customer-journey-agent.md) |
| User Story Agent | Produce testable stories with acceptance criteria | [user-story-agent.md](../../../ai/agents/product/user-story-agent.md) |
| MVP Planning Agent | Define the smallest product that solves the core problem | [mvp-planning-agent.md](../../../ai/agents/product/mvp-planning-agent.md) |

## Collaboration map

Discovery ([Customer Journey](../../../ai/agents/product/customer-journey-agent.md)) → definition
([Product Manager](../../../ai/agents/product/product-manager.md) + [User Story](../../../ai/agents/product/user-story-agent.md)) →
prioritization ([Feature Prioritization](../../../ai/agents/product/feature-prioritization-agent.md)) →
scope ([MVP Planning](../../../ai/agents/product/mvp-planning-agent.md)) → handoff to
[engineering](../engineering/README.md).
