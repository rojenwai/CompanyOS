# Hardware - Tools

Tooling is a means, not an end. Categories used here:

| Category | Examples |
|---|---|
| Firmware | Embedded toolchains, HIL rigs, debuggers |
| Electrical | EDA/PCB, simulation, instruments |
| Mechanical | CAD, FEA/thermal, prototyping |
| Test & fleet | Test chambers, fleet telemetry, OTA |

Tool choices follow the [build vs. buy framework](../engineering/decision-frameworks.md).
