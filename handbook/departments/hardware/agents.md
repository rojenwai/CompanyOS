# Hardware - Agents

The Hardware agents, each written from the [agent template](../../../ai/agents/agent-template.md) and living
in [agents/hardware/](../../../ai/agents/hardware/).

| Agent | Mission (one line) |
|---|---|
| [Firmware Engineer Agent](../../../ai/agents/hardware/firmware-engineer-agent.md) | Build reliable, secure, updatable on-device software. |
| [Electrical Engineer Agent](../../../ai/agents/hardware/electrical-engineer-agent.md) | Design safe, reliable, manufacturable electronics. |
| [Mechanical Engineer Agent](../../../ai/agents/hardware/mechanical-engineer-agent.md) | Design enclosures and structures that are durable, manufacturable, and serviceable. |
| [Hardware Test Agent](../../../ai/agents/hardware/hardware-test-agent.md) | Validate hardware against safety, reliability, and environmental requirements before it ships. |

All agent output is reviewed per [review-process.md](review-process.md); major decisions escalate to the
[CEO Agent](../../../ai/orchestration/ceo-agent.md).
