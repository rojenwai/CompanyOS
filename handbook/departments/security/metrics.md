# Security - Metrics

**North-star:** Zero customer-impacting security incidents, with shrinking attack surface.

**Posture** - open criticals, attack surface, control coverage

**Response** - mean time to remediate, mean time to detect, SLA attainment

**Assurance** - pen-test findings trend, audit findings, secret-scan hits

Metrics are surfaced on the [dashboards](dashboards.md) and feed the
[Continuous Improvement Engine](../../../ai/orchestration/continuous-improvement-engine.md).
