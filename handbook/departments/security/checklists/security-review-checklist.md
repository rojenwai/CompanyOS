# Security Review Checklist

- [ ] Threat model completed and current
- [ ] No secrets in code, config, or logs
- [ ] Input validated, output encoded, queries parameterized
- [ ] AuthZ/authN enforced (least privilege)
- [ ] Dependencies scanned; no known criticals
- [ ] Encryption in transit and at rest
- [ ] SAST/DAST run and highs resolved
- [ ] Security logging in place (no PII)
