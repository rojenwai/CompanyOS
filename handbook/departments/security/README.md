# Security

Security protects the company, its products, and its customers' data from threats. It owns threat modeling, secure architecture, penetration testing, vulnerability management, privacy, and compliance. It reports to the [Chief Security Officer](../../company/roles.md) and its reviewer can block releases that cannot be auto-overridden.

## Interfaces

Security sets standards that [engineering](../engineering/security-standards.md) and [ai](../ai-engineering/ai-security.md) implement, gates releases via the [Security Reviewer](../../../ai/orchestration/security-reviewer.md), drives incident response with [post-launch](../post-launch/incident-response.md), and works with [legal](../legal/README.md) on compliance and privacy.
