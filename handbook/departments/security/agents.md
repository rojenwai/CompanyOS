# Security - Agents

The Security agents, each written from the [agent template](../../../ai/agents/agent-template.md) and living
in [agents/security/](../../../ai/agents/security/).

| Agent | Mission (one line) |
|---|---|
| [Security Architect Agent](../../../ai/agents/security/security-architect-agent.md) | Threat-model systems and design controls so weaknesses are prevented by design. |
| [Penetration Testing Agent](../../../ai/agents/security/penetration-testing-agent.md) | Find exploitable weaknesses before attackers do. |
| [Vulnerability Management Agent](../../../ai/agents/security/vulnerability-management-agent.md) | Continuously find, triage, and drive remediation of vulnerabilities across the stack. |
| [Compliance & Privacy Agent](../../../ai/agents/security/compliance-privacy-agent.md) | Keep the company compliant and customer data protected by design. |

All agent output is reviewed per [review-process.md](review-process.md); major decisions escalate to the
[CEO Agent](../../../ai/orchestration/ceo-agent.md).
