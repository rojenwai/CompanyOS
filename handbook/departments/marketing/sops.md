# Marketing - Standard Operating Procedures

### SOP-01 - Position a product

1. Define the target segment and their problem
2. Articulate the unique value and proof
3. Write the messaging framework
4. Validate with customers
5. Roll out across channels

### SOP-02 - Run a marketing experiment

1. State the hypothesis and metric
2. Bound the budget and time
3. Run and measure
4. Decide scale/kill
5. Record the learning
