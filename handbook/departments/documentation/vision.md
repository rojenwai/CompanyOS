# Documentation - Vision

- Horizon 1 - Every product and API has complete, accurate docs; internal knowledge has owners.
- Horizon 2 - Docs drift is detected automatically and fixed fast ([doc drift check](../post-launch/automation.md)).
- Horizon 3 - Docs that partly write and update themselves from the source of truth.

The end state: no one is blocked because the answer was missing, wrong, or unfindable.
