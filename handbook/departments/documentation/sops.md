# Documentation - Standard Operating Procedures

### SOP-01 - Publish documentation

1. Confirm accuracy against the product
2. Structure for the reader's task
3. Add examples
4. Assign owner and freshness date
5. Publish and index

### SOP-02 - Audit documentation freshness

1. Pull the freshness register
2. Flag stale and orphaned docs
3. Assign refresh or retirement
4. Verify updates
5. Report coverage
