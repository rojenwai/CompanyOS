# Documentation - Roles

| Role | Accountable for |
|---|---|
| Head of Documentation (COO domain) | Docs strategy, information architecture, standards |
| Technical Writer | Product docs, guides, tutorials |
| API Documentation Specialist | Reference docs, examples, changelogs |
| Knowledge Manager | Internal knowledge, ownership, freshness |

Human roles are amplified by the [department agents](agents.md). Every role follows the
[principles](principles.md) and the [review process](review-process.md).
