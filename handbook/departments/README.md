# Departments

Twenty self-contained operating manuals. Every department follows the **same shape** (the standard 18
files + `templates/` `checklists/` `examples/`), so once you know one, you know them all — see
[STRUCTURE.md](../../STRUCTURE.md#standard-department-structure).

Each department's AI agents live in [ai/agents/&lt;department&gt;/](../../ai/agents/README.md).

**Customizing:** delete the departments you don't need (and their agent folder), keep the ones you do, and
add new ones from [templates/department-template/](../templates/department-template/). See
[CUSTOMIZE.md](../../CUSTOMIZE.md).
