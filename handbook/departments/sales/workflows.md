# Sales - Workflows

The core repeatable processes this department runs.

### Run a deal

1. Receive a qualified opportunity
2. Discover the problem and success criteria
3. Demonstrate value (Sales Engineering)
4. Propose and negotiate on value
5. Close with legal/finance
6. Hand off to [customer-success](../customer-success/README.md)

### Build the sales motion

1. Founder-led sell to learn the ICP and message
2. Document what works
3. Define stages and qualification
4. Train and enable
5. Instrument conversion and forecast
