# Sales - Tools

Tooling is a means, not an end. Categories used here:

| Category | Examples |
|---|---|
| CRM | Pipeline and relationship management |
| Prospecting | Enrichment, sequencing |
| Selling | Demo/POC environments, proposal/CPQ |
| Enablement | Playbooks, call analytics |

Tool choices follow the [build vs. buy framework](../engineering/decision-frameworks.md).
