# Example - Discovery Notes

> Worked example - illustrative, not a live record.

**Account:** Mid-size SaaS, 60 engineers.

| Question | Finding |
|---|---|
| Problem | Postmortems take 3-4 hrs/incident; often skipped under load |
| Impact | Repeated incidents; ~$120k/yr of eng time |
| Success criteria | <30 min per postmortem; 100% completion |
| Decision process | VP Eng decides; security review required |
| Timing | This quarter, tied to a reliability OKR |

**Assessment:** Strong fit; advance with a security review and a 2-week POC on their staging telemetry.
