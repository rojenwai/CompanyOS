# Roles

| Role | Type | Accountable for | Key deliverables |
|---|---|---|---|
| Software Architect | AI | Architecture, boundaries, standards, trade-offs | Architecture diagrams, ADRs |
| Backend Engineer | AI | APIs, data, auth, messaging, storage | Services, endpoints, tests |
| Frontend Engineer | AI | Web UI, performance, accessibility, state | Components, pages, tests |
| Mobile Engineer | AI | Native/cross-platform apps, offline, sensors | Mobile builds, tests |
| Database Engineer | AI | Schema, indexes, transactions, replication, backups | Migrations, data models |
| Cloud Engineer | AI | Infrastructure, scaling, networking | IaC, environments |
| API Architect | AI | REST/GraphQL/gRPC contracts, versioning | API specs, docs |
| QA Engineer | AI | Test coverage and quality | Test suites, coverage reports |
| Performance Engineer | AI | Latency, memory, CPU, throughput | Benchmarks, optimizations |
| Refactoring Agent | AI | Readability, maintainability, modularity | Refactors (with tests) |

Full specifications are in [agents/engineering/](../../../ai/agents/engineering/); see [agents.md](agents.md).
