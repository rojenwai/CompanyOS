# Example - Role Scorecard

> Worked example - illustrative, not a live record.

**Role:** Senior Backend Engineer.

**Mission:** Own the reliability and scalability of the core API for the next 12 months.

| Competency | Signal of the bar |
|---|---|
| System design | Has designed and run a system at our scale or above |
| Code quality | Sets standards others follow; strong on testing |
| Ownership | Drives incidents and debt repayment without being asked |
| Collaboration | Raises the team; reviews generously and rigorously |

**Outcomes (12 mo):** API p99 within SLO; on-call load down 30%; two engineers leveled up by their mentoring.
