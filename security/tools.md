# Security - Tools

Tooling is a means, not an end. Categories used here:

| Category | Examples |
|---|---|
| Scanning | SCA, SAST, DAST, secret scanners |
| Offensive | Exploitation frameworks, fuzzers (authorized) |
| Secrets & access | Vault, IAM, SSO |
| GRC | Compliance and policy management |

Tool choices follow the [build vs. buy framework](../engineering/decision-frameworks.md).
