# Security - Principles

Non-negotiable. Every piece of work is judged against these.

1. **Secure by default.** The safe path is the easy path; insecure requires justification.
2. **Least privilege.** Every actor and agent gets the minimum access it needs.
3. **Defense in depth.** No single control is trusted alone.
4. **Assume breach.** Design for containment and fast detection, not just prevention.
5. **No secrets in code, ever.** Secrets live in a vault, never in Git ([security-standards](../engineering/security-standards.md)).
6. **Privacy is security.** Minimize data collected; protect what is kept.
7. **Blameless response.** Incidents teach; people are not punished for reporting ([incident-response](../post-launch/incident-response.md)).
