# Security - Roles

| Role | Accountable for |
|---|---|
| Chief Security Officer | Security posture, risk decisions, final security review |
| Security Architect | Threat models, secure design, control selection |
| Penetration Tester | Offensive testing, red-team exercises |
| Vulnerability Manager | Scanning, triage, patch orchestration |
| Compliance & Privacy Lead | Frameworks, audits, data-protection |

Human roles are amplified by the [department agents](agents.md). Every role follows the
[principles](principles.md) and the [review process](review-process.md).
