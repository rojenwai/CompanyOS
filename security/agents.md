# Security - Agents

The Security agents, each written from the [agent template](../agents/agent-template.md) and living
in [agents/security/](../agents/security/).

| Agent | Mission (one line) |
|---|---|
| [Security Architect Agent](../agents/security/security-architect-agent.md) | Threat-model systems and design controls so weaknesses are prevented by design. |
| [Penetration Testing Agent](../agents/security/penetration-testing-agent.md) | Find exploitable weaknesses before attackers do. |
| [Vulnerability Management Agent](../agents/security/vulnerability-management-agent.md) | Continuously find, triage, and drive remediation of vulnerabilities across the stack. |
| [Compliance & Privacy Agent](../agents/security/compliance-privacy-agent.md) | Keep the company compliant and customer data protected by design. |

All agent output is reviewed per [review-process.md](review-process.md); major decisions escalate to the
[CEO Agent](../orchestration/ceo-agent.md).
