Design turns validated problems and requirements into experiences that are clear, consistent, and
accessible. It reports to the [Chief Designer](../company/roles.md) and partners with
[product](../product/README.md) and [engineering](../engineering/README.md).

## Design organization

| Function | Produces |
|---|---|
| **UX Research** | Behavior, needs, pain points, mental models, accessibility findings — see [ux-research.md](ux-research.md) |
| **UX Design** | User flows, information architecture, wireframes, prototypes, interaction models |
| **UI Design** | Layout, typography, spacing, icons, illustration, color system, components, themes |

## Design principles

Consistency · clarity · accessibility · efficiency · feedback · predictability · minimal cognitive
load · visual hierarchy.
