# Design - Playbooks

Situational guides. Each names the trigger and the steps.

### Usability test reveals confusion

1. Locate the friction point in the flow
2. Diagnose root cause (labeling, hierarchy, feedback)
3. Redesign the smallest change that fixes it
4. Re-test
5. Update the design system if systemic

### Accessibility audit fails before launch

1. Triage issues by severity
2. Fix criticals immediately
3. Re-audit with assistive tech
4. Block launch until criticals clear
5. Add the case to the accessibility checklist
