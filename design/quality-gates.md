# Design - Quality Gates

Work does not progress until the relevant gate passes. These implement the company
[quality gates](../governance/quality-gates.md).

| Gate | Must pass |
|---|---|
| Design-system gate | Built from documented components and tokens |
| Accessibility gate | Meets WCAG; assistive-tech tested |
| States gate | Loading, empty, and error states designed |
| Validation gate | Significant changes usability-tested |
