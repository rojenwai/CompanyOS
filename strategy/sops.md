# Strategy - Standard Operating Procedures

### SOP-01 - Build a business model

1. Map value per segment
2. Choose model and pricing
3. Model unit economics
4. Stress-test assumptions
5. Record the decision

### SOP-02 - Plan a go-to-market

1. Pick the segment and positioning
2. Select channels and motion
3. Set funnel targets
4. Coordinate the launch
5. Instrument and review
