# Strategy - Mission

**Turn a good product into an enduring, defensible company.**

Engineering builds products; strategy builds the company. Every strategic decision must create meaningful customer value, strengthen competitive advantage, and improve long-term company value. Never start with technology - start with a problem.

Priority order when they conflict:

1. **Customer value** - does this create real, measurable value?
2. **Defensibility** - does it strengthen a durable moat?
3. **Company value** - does it improve the long-term value of the business?
4. **Scalability** - can it become a global business?
