# Finance - Tools

Tooling is a means, not an end. Categories used here:

| Category | Examples |
|---|---|
| Accounting | ERP/accounting systems |
| Planning | Financial modeling, FP&A tooling |
| Revenue | Billing, payments, revenue recognition |
| Cost | Cloud/AI cost management, BI |

Tool choices follow the [build vs. buy framework](../engineering/decision-frameworks.md).
