# Finance - Standard Operating Procedures

### SOP-01 - Run the monthly close

1. Ingest all feeds
2. Reconcile accounts
3. Book accruals
4. Produce statements
5. Update forecast and runway
6. Distribute the package

### SOP-02 - Approve a purchase

1. Confirm the budget line and value
2. Check runway impact
3. Obtain the required approval
4. Record the commitment
5. Track against budget
