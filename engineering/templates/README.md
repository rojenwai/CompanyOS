# Engineering Templates

Engineering-specific templates. Company-wide document templates (ADR, RFC, security review, release
checklist) live in [templates/documents/](../../templates/documents/) — use those directly:

- [ADR](../../templates/documents/adr.md) — record an architecture decision.
- [RFC](../../templates/documents/rfc.md) — propose a significant change.
- [Security review](../../templates/documents/security-review.md).
- [Release checklist](../../templates/documents/release-checklist.md).

Add engineering-only templates (e.g. a service scaffold README) here as they are created.
