# Operations - Metrics

**North-star:** Predictable, low-friction execution across the company.

**Delivery** - on-time delivery rate, cycle time, blocker resolution time

**Process** - manual steps eliminated, process adoption, status freshness

**Operations** - vendor spend vs. budget, documentation freshness, asset accuracy

Metrics are surfaced on the [dashboards](dashboards.md) and feed the
[Continuous Improvement Engine](../orchestration/continuous-improvement-engine.md).
