# Sales - Playbooks

Situational guides. Each names the trigger and the steps.

### A deal stalls

1. Confirm the problem is real and prioritized
2. Identify the true blocker (budget, authority, timing)
3. Re-anchor on value and cost of inaction
4. Offer a smaller first step if fit is real
5. Qualify out if it is not

### A customer asks for an unbuilt feature

1. Understand the underlying need
2. Check the [roadmap](../product/roadmap.md)
3. Never promise unbuilt features as committed
4. Route the signal to [product](../post-launch/feature-requests.md)
5. Set honest expectations
