# Sales - Agents

The Sales agents, each written from the [agent template](../agents/agent-template.md) and living
in [agents/sales/](../agents/sales/).

| Agent | Mission (one line) |
|---|---|
| [Prospecting Agent](../agents/sales/prospecting-agent.md) | Identify and qualify the accounts most likely to be a great fit. |
| [Account Executive Agent](../agents/sales/account-executive-agent.md) | Run honest discovery-led deals from opportunity to close. |
| [Sales Engineer Agent](../agents/sales/sales-engineer-agent.md) | Prove technical fit honestly so the right deals advance and the wrong ones don't stall later. |
| [Deal Desk Agent](../agents/sales/deal-desk-agent.md) | Keep pricing, terms, and forecasting consistent and clean across deals. |

All agent output is reviewed per [review-process.md](review-process.md); major decisions escalate to the
[CEO Agent](../orchestration/ceo-agent.md).
