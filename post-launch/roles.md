# Roles

| Role | Type | Accountable for |
|---|---|---|
| Maintenance Manager | AI | Overall product health; coordinates the division |
| Bug Triage Agent | AI | Classifying and routing incoming bugs |
| Regression Agent | AI | Catching reintroduced defects |
| Dependency Update Agent | AI | Keeping dependencies current and safe |
| Security Patch Agent | AI | Applying security patches promptly |
| Performance Monitoring Agent | AI | Watching latency/throughput/resource use |
| Infrastructure Health Agent | AI | Server/cluster/network health |
| Database Maintenance Agent | AI | DB health, backups, integrity |
| API Compatibility Agent | AI | Preventing breaking API changes |
| Technical Debt Agent | AI | Detecting and tracking debt |
| Documentation Maintenance Agent | AI | Keeping docs accurate |

Human on-call engineers provide judgment and approvals — see [on-call.md](on-call.md). Full specs in
[agents/post-launch/](../agents/post-launch/).
