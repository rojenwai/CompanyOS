# Marketplace - Added Standards

These extend the core [standards](../../standards/README.md) with what this company type specifically requires.

1. Track both sides' health separately; liquidity (match rate, time-to-match) is the north-star.
2. Payments and payouts are accurate, timely, and fraud-screened ([finance](../../finance/README.md)).
3. Trust & safety (identity, ratings, dispute resolution) is a core product surface, not an afterthought.
4. Cold-start is solved deliberately for each new market/segment.
