# Marketplace - Added Workflow

Layered on top of the core [SDLC](../../workflows/sdlc.md), this domain adds:

```
Seed supply -> seed demand -> matching
  -> transaction + payments/payouts
  -> ratings + trust & safety -> liquidity growth
  -> expand to new markets (repeat cold-start)
```

## How it maps to core

- Matching and liquidity metrics use [data](../../data/README.md).
- Payments use [finance](../../finance/README.md); disputes use [operations](../../operations/README.md).
- Trust & safety uses [security](../../security/README.md) and [operations](../../operations/README.md).
- Two-sided growth uses [marketing](../../marketing/README.md) and [strategy GTM](../../strategy/go-to-market.md).
