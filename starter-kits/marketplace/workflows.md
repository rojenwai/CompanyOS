# Marketplace - Added Workflow

Layered on top of the core [SDLC](../../handbook/workflows/sdlc.md), this domain adds:

```
Seed supply -> seed demand -> matching
  -> transaction + payments/payouts
  -> ratings + trust & safety -> liquidity growth
  -> expand to new markets (repeat cold-start)
```

## How it maps to core

- Matching and liquidity metrics use [data](../../handbook/departments/data/README.md).
- Payments use [finance](../../handbook/departments/finance/README.md); disputes use [operations](../../handbook/departments/operations/README.md).
- Trust & safety uses [security](../../handbook/departments/security/README.md) and [operations](../../handbook/departments/operations/README.md).
- Two-sided growth uses [marketing](../../handbook/departments/marketing/README.md) and [strategy GTM](../../handbook/departments/strategy/go-to-market.md).
