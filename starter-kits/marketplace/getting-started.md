# Marketplace - Getting Started

1. Adopt the core; validate both sides' problems.
2. Pick a beachhead market and a cold-start plan for each side.
3. Instrument liquidity from day one ([data](../../handbook/departments/data/README.md)).
4. Stand up payments/payouts ([finance](../../handbook/departments/finance/README.md)) and trust & safety ([security](../../handbook/departments/security/README.md)).
5. Ship the MVP for one liquid market; prove the transaction loop.
6. Expand market by market, re-solving cold-start each time.

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
