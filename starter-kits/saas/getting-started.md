# SaaS - Getting Started

1. Adopt the core ([company](../../company/README.md), [governance](../../governance/README.md), [kernel](../../orchestration/README.md)).
2. Validate the problem and pick the [business model & pricing](../../strategy/business-model.md).
3. Design multi-tenant architecture ([engineering](../../engineering/organization.md)) and billing.
4. Activate [customer-success](../../customer-success/README.md) and instrument activation/churn early.
5. Ship the MVP ([build-an-MVP playbook](../../playbooks/building-an-mvp.md)); publish an SLA.
6. Run the retention loop ([continuous improvement](../../post-launch/continuous-improvement.md)).

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
