# SaaS - Getting Started

1. Adopt the core ([company](../../handbook/company/README.md), [governance](../../handbook/governance/README.md), [kernel](../../ai/orchestration/README.md)).
2. Validate the problem and pick the [business model & pricing](../../handbook/departments/strategy/business-model.md).
3. Design multi-tenant architecture ([engineering](../../handbook/departments/engineering/organization.md)) and billing.
4. Activate [customer-success](../../handbook/departments/customer-success/README.md) and instrument activation/churn early.
5. Ship the MVP ([build-an-MVP playbook](../../handbook/playbooks/building-an-mvp.md)); publish an SLA.
6. Run the retention loop ([continuous improvement](../../handbook/departments/post-launch/continuous-improvement.md)).

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
