# SaaS - Added Standards

These extend the core [standards](../../standards/README.md) with what this company type specifically requires.

1. Multi-tenancy isolates every tenant's data by default; cross-tenant access is a critical bug.
2. Every plan has metered/subscription billing that reconciles to [finance](../../finance/README.md).
3. Public uptime SLA with error budgets ([SLA](../../post-launch/sla.md)); breaches trigger incident response.
4. Self-serve onboarding drives users to first value without human touch where possible.
