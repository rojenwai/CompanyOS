# Robotics - Getting Started

1. Adopt the core; define the task and the safety envelope first.
2. Activate [hardware](../../handbook/departments/hardware/README.md) and design for safety and serviceability.
3. Build perception/control with sim + hardware-in-the-loop [evaluation](../../handbook/departments/ai-engineering/evaluation-and-testing.md).
4. Run integrated safety testing ([Hardware Test Agent](../../ai/agents/hardware/hardware-test-agent.md)).
5. Deploy a pilot fleet with OTA and telemetry.
6. Operate the fleet via [post-launch](../../handbook/departments/post-launch/README.md); learn and iterate.

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
