# Developer Tools - Added / Activated

Beyond the core [agents](../../ai/agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../ai/agents/agent-template.md).

## Departments to activate first

- [documentation/](../../handbook/departments/documentation/README.md) - docs ARE the product surface
- [engineering/](../../handbook/departments/engineering/api-standards.md) - API-first, stability, versioning
- [marketing/](../../handbook/departments/marketing/README.md) - developer relations and content
- [post-launch/](../../handbook/departments/post-launch/README.md) - API compatibility and deprecation discipline

## Emphasis

Emphasize the [API Documentation Agent](../../ai/agents/documentation/api-documentation-agent.md), [API Architect](../../ai/agents/engineering/api-architect.md), and [API Compatibility Agent](../../ai/agents/post-launch/api-compatibility-agent.md).
