# Drone / UAV - Added / Activated

Beyond the core [agents](../../ai/agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../ai/agents/agent-template.md).

## Departments to activate first

- [hardware/](../../handbook/departments/hardware/README.md) - airframe, propulsion, avionics, firmware
- [ai/](../../handbook/departments/ai-engineering/README.md) - autonomy, navigation, obstacle avoidance
- [legal/](../../handbook/departments/legal/README.md) - aviation regulation (e.g. FAA/EASA), airspace compliance
- [security/](../../handbook/departments/security/README.md) - command link security, geofencing, safety

## Emphasis

Emphasize the [Hardware Test Agent](../../ai/agents/hardware/hardware-test-agent.md), AI autonomy models, and the [Compliance Agent](../../ai/agents/legal/compliance-agent.md) for airspace regulation.
