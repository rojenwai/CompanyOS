# Drone / UAV - Added / Activated

Beyond the core [agents](../../agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../agents/agent-template.md).

## Departments to activate first

- [hardware/](../../hardware/README.md) - airframe, propulsion, avionics, firmware
- [ai/](../../ai/README.md) - autonomy, navigation, obstacle avoidance
- [legal/](../../legal/README.md) - aviation regulation (e.g. FAA/EASA), airspace compliance
- [security/](../../security/README.md) - command link security, geofencing, safety

## Emphasis

Emphasize the [Hardware Test Agent](../../agents/hardware/hardware-test-agent.md), AI autonomy models, and the [Compliance Agent](../../agents/legal/compliance-agent.md) for airspace regulation.
