# Drone / UAV - Added Standards

These extend the core [standards](../../standards/README.md) with what this company type specifically requires.

1. Regulatory compliance (airspace, certification) is a release gate owned by [legal](../../legal/README.md).
2. Fail-safe behavior (return-to-home, safe-land, geofencing) is mandatory and tested.
3. Autonomy is validated in simulation and controlled flight tests before general operation.
4. Command-and-control links are authenticated and encrypted ([security](../../security/README.md)).
