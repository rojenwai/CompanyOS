# Drone / UAV - Getting Started

1. Adopt the core; scope the mission and the regulatory environment early.
2. Engage [legal](../../legal/README.md) on airspace/certification before building far.
3. Activate [hardware](../../hardware/README.md); design fail-safe behavior in.
4. Develop autonomy with staged sim -> tethered -> controlled-flight [evaluation](../../ai/evaluation-and-testing.md).
5. Test geofencing and fail-safes exhaustively ([Hardware Test Agent](../../agents/hardware/hardware-test-agent.md)).
6. Operate under compliance with fleet monitoring and [incident response](../../post-launch/incident-response.md).

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
