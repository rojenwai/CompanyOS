# Research Lab - Getting Started

1. Adopt the core; frame the research questions that matter ([research framework](../../research/research-framework.md)).
2. Set up rigorous, reproducible experimentation ([ai evaluation](../../ai/evaluation-and-testing.md)).
3. Establish IP strategy early with [legal](../../legal/README.md).
4. Run internal peer review before external claims.
5. Decide publish vs. protect vs. productize per result.
6. Hand productizable findings to [product](../../product/README.md) via the [opportunity score](../../research/opportunity-scoring.md).
