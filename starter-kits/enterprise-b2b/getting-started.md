# Enterprise B2B - Getting Started

1. Adopt the core; validate the enterprise problem and buyer.
2. Stand up [security compliance](../../security/README.md) early (it gates deals).
3. Prepare standard [contracts](../../legal/README.md) (MSA, DPA) and a control library.
4. Build the sales-led motion ([sales](../../sales/README.md)) with solution engineering.
5. Design enterprise features: SSO, RBAC, audit logs, data residency.
6. Run enterprise [customer-success](../../customer-success/README.md) with QBRs and renewals.

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
