# Enterprise B2B - Added / Activated

Beyond the core [agents](../../agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../agents/agent-template.md).

## Departments to activate first

- [security/](../../security/README.md) - compliance (SOC 2, ISO), security questionnaires
- [legal/](../../legal/README.md) - MSAs, DPAs, procurement, negotiation
- [sales/](../../sales/README.md) - sales-led, solution engineering, deal desk
- [customer-success/](../../customer-success/README.md) - enterprise onboarding and QBRs

## Emphasis

Emphasize the [Compliance & Privacy Agent](../../agents/security/compliance-privacy-agent.md), [Contract Agent](../../agents/legal/contract-agent.md), and [Sales Engineer Agent](../../agents/sales/sales-engineer-agent.md).
