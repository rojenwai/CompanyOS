# Enterprise B2B - Added / Activated

Beyond the core [agents](../../ai/agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../ai/agents/agent-template.md).

## Departments to activate first

- [security/](../../handbook/departments/security/README.md) - compliance (SOC 2, ISO), security questionnaires
- [legal/](../../handbook/departments/legal/README.md) - MSAs, DPAs, procurement, negotiation
- [sales/](../../handbook/departments/sales/README.md) - sales-led, solution engineering, deal desk
- [customer-success/](../../handbook/departments/customer-success/README.md) - enterprise onboarding and QBRs

## Emphasis

Emphasize the [Compliance & Privacy Agent](../../ai/agents/security/compliance-privacy-agent.md), [Contract Agent](../../ai/agents/legal/contract-agent.md), and [Sales Engineer Agent](../../ai/agents/sales/sales-engineer-agent.md).
