# Enterprise B2B - Added Standards

These extend the core [standards](../../standards/README.md) with what this company type specifically requires.

1. Maintain audit-ready compliance evidence (e.g. SOC 2) continuously, not before each audit.
2. SSO, role-based access, and audit logs are table-stakes, not roadmap items.
3. Every enterprise contract passes [legal review](../../legal/review-process.md) with bounded liability.
4. Security questionnaires are answered from a maintained, accurate control library.
