# IoT Platform - Getting Started

1. Adopt the core; define the device, connectivity, and data model.
2. Activate [hardware](../../hardware/README.md) and secure device provisioning.
3. Build reliable telemetry pipelines ([data](../../data/README.md)).
4. Design fleet OTA (canary + rollback) and offline behavior.
5. Stand up fleet infrastructure ([devops](../../devops/README.md)).
6. Operate and monitor the fleet via [post-launch](../../post-launch/README.md).

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
