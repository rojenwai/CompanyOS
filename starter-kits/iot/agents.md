# IoT Platform - Added / Activated

Beyond the core [agents](../../ai/agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../ai/agents/agent-template.md).

## Departments to activate first

- [hardware/](../../handbook/departments/hardware/README.md) - device and firmware
- [data/](../../handbook/departments/data/README.md) - ingestion pipelines and device analytics at scale
- [security/](../../handbook/departments/security/README.md) - device identity, connectivity security, fleet
- [devops/](../../handbook/departments/devops/README.md) - fleet-scale infrastructure and OTA

## Emphasis

Emphasize the [Data Engineer Agent](../../ai/agents/data/data-engineer-agent.md) for telemetry pipelines, [Firmware Engineer Agent](../../ai/agents/hardware/firmware-engineer-agent.md), and device security.
