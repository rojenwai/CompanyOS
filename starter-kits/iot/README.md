# Starter Kit: IoT Platform

A company operating fleets of connected devices feeding cloud services and data pipelines.

## What it inherits (unchanged)

- The full [company identity](../../company/), [governance](../../governance/), and [kernel](../../orchestration/) ([memory](../../memory/), [agents](../../agents/)).
- The [engineering](../../engineering/), [product](../../product/), and [post-launch](../../post-launch/) departments and every other core department.
- The [SDLC](../../workflows/sdlc.md), [standards](../../standards/), [templates](../../templates/), and [strategy](../../strategy/).

## What it adds / emphasizes

| Delta | Where |
|---|---|
| Domain agents & activated departments | [agents.md](agents.md) |
| Domain standards | [standards.md](standards.md) |
| Domain delivery lifecycle | [workflows.md](workflows.md) |
| How to stand this up | [getting-started.md](getting-started.md) |

## Why these deltas

IoT combines device fleets, connectivity, cloud, and data at scale. This kit activates hardware, data pipelines, device security, and fleet-scale devops.

## Swapping this kit

To build a different company type, copy [starter-kits/ai-startup/](../ai-startup/) or this folder and replace
the domain deltas with your own - the inheritance model is identical.
