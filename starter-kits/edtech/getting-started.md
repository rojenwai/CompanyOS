# Ed Tech - Getting Started

1. Adopt the core; validate the learning problem with learners **and** educators.
2. Design the pedagogy before the interface.
3. Build accessibility ([WCAG](../../handbook/departments/design/accessibility.md)) and data minimisation in from the start.
4. Clear student-privacy obligations with [legal](../../handbook/departments/legal/README.md) before collecting anything.
5. Pilot in real classrooms; run an efficacy study against a baseline.
6. Sell to institutions on evidence, then support rollout with [customer-success](../../handbook/departments/customer-success/README.md).

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
