# Ed Tech - Added Workflow

Layered on top of the core [SDLC](../../handbook/workflows/sdlc.md), this domain adds:

```
Learning problem + learner research
  -> pedagogy & instructional design
  -> accessibility + privacy by design
  -> build -> classroom pilot
  -> efficacy study (does it teach?) [gate]
  -> institutional rollout -> outcome reporting
```

## How it maps to core

- Efficacy studies use [research](../../handbook/departments/research/README.md) and [data](../../handbook/departments/data/README.md).
- Accessibility and age-appropriate UX use [design](../../handbook/departments/design/README.md).
- Student-data privacy uses [legal](../../handbook/departments/legal/README.md) and [security](../../handbook/departments/security/README.md).
- Institutional sales and rollout use [sales](../../handbook/departments/sales/README.md) and [customer-success](../../handbook/departments/customer-success/README.md).
