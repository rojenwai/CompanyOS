# Gaming - Added / Activated

Beyond the core [agents](../../ai/agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../ai/agents/agent-template.md).

## Departments to activate first

- [design/](../../handbook/departments/design/README.md) - game feel, onboarding, playtesting
- [data/](../../handbook/departments/data/README.md) - telemetry, retention cohorts, balance
- [post-launch/](../../handbook/departments/post-launch/README.md) - live-ops, content cadence, incidents
- [marketing/](../../handbook/departments/marketing/README.md) - community, launch, creators

## Emphasis

Emphasize design agents for playtesting, the [Data Scientist Agent](../../ai/agents/data/data-scientist-agent.md) for retention and balance, the [Performance Engineer](../../ai/agents/engineering/performance-engineer.md) for frame budgets, and [customer-success](../../handbook/departments/customer-success/README.md) for community and player support.
