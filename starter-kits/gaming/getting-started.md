# Gaming - Getting Started

1. Adopt the core; state the core loop you believe is fun.
2. Prototype fast and playtest ruthlessly - kill it if the loop is not fun.
3. Set performance budgets and a quality bar with a vertical slice.
4. Instrument retention cohorts before soft launch ([data](../../handbook/departments/data/README.md)).
5. Soft launch, measure D1/D7/D30, and iterate on the loop - not just content.
6. Run live-ops with a committed cadence and ethical monetization.

> Everything not listed here comes from core Company OS - this kit only adds the domain-specific deltas.
