# Climate Tech - Added / Activated

Beyond the core [agents](../../ai/agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../ai/agents/agent-template.md).

## Departments to activate first

- [research/](../../handbook/departments/research/README.md) - science, first principles, [TRL](../../handbook/departments/research/technology-readiness.md)
- [hardware/](../../handbook/departments/hardware/README.md) - physical systems, safety, manufacturability
- [data/](../../handbook/departments/data/README.md) - measurement, reporting & verification (MRV)
- [strategy/](../../handbook/departments/strategy/README.md) - grants, project finance, offtake agreements

## Emphasis

Emphasize the research agents (especially [Opportunity Ranking](../../ai/agents/research/opportunity-ranking-agent.md)) with honest [TRL](../../handbook/departments/research/technology-readiness.md) assessment, the [Hardware Test Agent](../../ai/agents/hardware/hardware-test-agent.md), and the [Data Engineer Agent](../../ai/agents/data/data-engineer-agent.md) for MRV pipelines.
