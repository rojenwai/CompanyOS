# Climate Tech - Added Workflow

Layered on top of the core [SDLC](../../handbook/workflows/sdlc.md), this domain adds:

```
Scientific problem + first principles (TRL 1-3)
  -> lab proof of concept [TRL gate]
  -> pilot in a relevant environment (TRL 4-6, the valley of death)
  -> demonstration at scale (TRL 7-8)
  -> deployment (TRL 9) + MRV instrumentation
  -> verified impact reporting -> scale / project finance
```

## How it maps to core

- Science and TRL assessment use [research](../../handbook/departments/research/README.md).
- Physical systems, safety, and manufacturability use [hardware](../../handbook/departments/hardware/README.md).
- MRV pipelines and verified reporting use [data](../../handbook/departments/data/README.md).
- Grants, project finance, and offtakes use [strategy](../../handbook/departments/strategy/README.md) and [finance](../../handbook/departments/finance/README.md).
