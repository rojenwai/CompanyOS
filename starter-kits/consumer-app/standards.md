# Consumer App - Added Standards

These extend the core [standards](../../standards/README.md) with what this company type specifically requires.

1. Retention (D1/D7/D30 cohorts) is the north-star; vanity installs are an anti-metric.
2. Every growth loop is instrumented and measured; scale only proven loops.
3. Mobile performance budgets (startup time, crash-free rate) are enforced.
4. Experimentation (A/B) gates significant UX and growth changes ([product analytics](../../product/analytics.md)).
