# Health Tech - Added / Activated

Beyond the core [agents](../../ai/agents/README.md), this kit leans on these departments and agents.
Write any new agents from the [agent template](../../ai/agents/agent-template.md).

## Departments to activate first

- [legal/](../../handbook/departments/legal/README.md) - regulatory classification, HIPAA/GDPR-health, consent
- [security/](../../handbook/departments/security/README.md) - PHI protection, access, audit trails
- [research/](../../handbook/departments/research/README.md) - clinical evidence and validation
- [documentation/](../../handbook/departments/documentation/README.md) - regulatory and quality documentation

## Emphasis

Emphasize the [Compliance Agent](../../ai/agents/legal/compliance-agent.md), [Privacy & Terms Agent](../../ai/agents/legal/privacy-terms-agent.md), the research agents for clinical evidence, and - for any diagnostic model - the [AI Evaluation Agent](../../ai/agents/ai-engineering/ai-evaluation-agent.md).
