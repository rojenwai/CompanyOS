# DevOps - Roles

| Role | Accountable for |
|---|---|
| DevOps / Platform Lead | Delivery and infrastructure strategy |
| CI/CD Engineer | Pipelines, build, test automation, release engineering |
| Platform / Infrastructure Engineer | IaC, environments, networking, cost |
| Site Reliability Engineer | SLOs, capacity, reliability, incident tooling |

Human roles are amplified by the [department agents](agents.md). Every role follows the
[principles](principles.md) and the [review process](review-process.md).
