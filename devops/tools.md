# DevOps - Tools

Tooling is a means, not an end. Categories used here:

| Category | Examples |
|---|---|
| CI/CD | Pipeline systems, artifact registries |
| IaC | Infrastructure-as-code tools |
| Delivery | Feature flags, deploy orchestration |
| Reliability | Observability, autoscaling, chaos testing |

Tool choices follow the [build vs. buy framework](../engineering/decision-frameworks.md).
