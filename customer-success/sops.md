# Customer Success - Standard Operating Procedures

### SOP-01 - Onboard a new customer

1. Review success criteria from sales
2. Configure and train
3. Drive to first value
4. Confirm activation
5. Set the health baseline and assign a CSM

### SOP-02 - Run a renewal

1. Start 90 days out
2. Quantify delivered value
3. Confirm health and fit
4. Present the value case
5. Close and record; flag expansion
