# Department Memory

**Scope:** one department, long-lived.

## What's stored
A department's operating knowledge: its workflows, SOPs, playbooks, standards, and accumulated know-how — backed by the department's own folder (e.g. [engineering/](../engineering/)).

## Lifetime
Permanent; versioned with the department.

## Writers / readers
Maintained by department agents; read by any agent operating in that domain.

## Relationship to company memory
Specializes [company memory](company-memory.md); where they conflict, company-wide standards win unless a documented exception exists.
