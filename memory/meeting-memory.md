# Meeting Memory

**Scope:** decisions and action items from meetings, long-lived.

## What's stored
[Meeting notes](../templates/documents/meeting-notes.md): decisions, owners, action items, and follow-ups.

## Lifetime
Retained; action items tracked to closure.

## Writers / readers
Written from meeting notes; decisions are promoted to [decision-memory.md](decision-memory.md) and action items to the relevant project.

## Rule
A meeting without recorded decisions and action items is not complete.
