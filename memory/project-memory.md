# Project Memory

**Scope:** a single project, across sessions.

## What's stored
Architecture · requirements · roadmap · tasks · design decisions · open risks — the durable state a project needs to be resumed by any agent.

## Lifetime
Lives for the life of the project; archived at [end-of-life](../post-launch/end-of-life.md).

## Writers / readers
Written by project agents via the [Memory Manager](../orchestration/memory-manager.md); read at task start to assemble context.

## Links
Architectural rationale → [architecture-memory.md](architecture-memory.md); decisions → [decision-memory.md](decision-memory.md).
