# User Memory

**Scope:** an individual user, long-lived.

## What's stored
Preferences · goals · previous work · constraints · long-term projects — the context that personalizes how agents serve a specific user.

## Lifetime
Persists across sessions; user-controlled.

## Writers / readers
Written on explicit user signals or confirmed inference; read to tailor responses.

## Privacy
Contains personal data — governed by [legal/privacy](../company/roles.md) and never logged to observability. See [retention-and-versioning.md](retention-and-versioning.md).
