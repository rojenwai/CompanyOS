# Data - Review Process

Data work is reviewed for correctness (do numbers reconcile and tests pass?), governance (is sensitive data protected?), and clarity (are definitions documented?). Analyses are reviewed for method; governance is reviewed with security and legal.

Reviews are independent - the [Reviewer](../orchestration/reviewer.md) never assumes the author is
correct. Security-sensitive work also requires the
[Security Reviewer](../orchestration/security-reviewer.md).
