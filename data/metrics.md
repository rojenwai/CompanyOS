# Data - Metrics

**North-star:** Trusted, timely data powering every decision - no dueling numbers.

**Reliability** - pipeline uptime, data freshness, quality test pass rate

**Trust** - metric definition coverage, definition consistency, self-serve adoption

**Governance** - data classified (%), access compliance, privacy incidents

Metrics are surfaced on the [dashboards](dashboards.md) and feed the
[Continuous Improvement Engine](../orchestration/continuous-improvement-engine.md).
