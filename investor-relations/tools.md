# Investor Relations - Tools

Tooling is a means, not an end. Categories used here:

| Category | Examples |
|---|---|
| Reporting | BI, investor-update tooling |
| Data room | Secure document sharing |
| Governance | Board portal, document management |
| Cap table | Cap-table and equity platform |

Tool choices follow the [build vs. buy framework](../engineering/decision-frameworks.md).
