# Investor Relations - Agents

The Investor Relations agents, each written from the [agent template](../agents/agent-template.md) and living
in [agents/investor-relations/](../agents/investor-relations/).

| Agent | Mission (one line) |
|---|---|
| [Investor Reporting Agent](../agents/investor-relations/investor-reporting-agent.md) | Produce accurate, consistent investor updates that reconcile to finance. |
| [Board Governance Agent](../agents/investor-relations/board-governance-agent.md) | Run clean board process - well-prepared meetings, accurate minutes, proper resolutions. |
| [Cap Table Agent](../agents/investor-relations/cap-table-agent.md) | Keep ownership records accurate and model the impact of financing and equity events. |

All agent output is reviewed per [review-process.md](review-process.md); major decisions escalate to the
[CEO Agent](../orchestration/ceo-agent.md).
